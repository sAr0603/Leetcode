## Solution :

```python
class Solution:
    def minPathSum(self, grid: List[List[int]]) -> int:
        n, m = len(grid), len(grid[0])

        for r in range(n):
            for c in range(m):
                u = {
                    grid[r - 1][c] if r else sys.maxsize,
                    grid[r][c - 1] if c else sys.maxsize
                }
                grid[r][c] += (min(u) if min(u) != sys.maxsize else 0)
        return grid[n - 1][m - 1]
```
